# unix-rs

## module structure
```bash
├── client
├── CMakeLists.txt
├── docs
├── lib
├── LICENSE
├── Makefile
├── README.md
├── server
├── unix-rs
└── unp-rs
```
* client: c++ module client
* CMakeLists.txt: c++ module cmake
* docs: documentation
* lib: c++ module lib
* LICENSE: license
* Makefile: global Makefile
* README.md: readme file
* server: c++ module server
* unix-rs: rust unix binding
* unp-rs: rust module
